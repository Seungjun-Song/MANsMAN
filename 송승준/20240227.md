- 데이터의 결측이 발생하거나 이상값이 발생할 경우 Machine Learning 등 데이터 분석 모델링 결과에 왜곡 또는 편향된 결과를 도래할 수 있다. 따라서 데이터 전처리와 EDA 과정을 통해서 분포를 파악하고 데이터를 다시 정제해서 예측되는 모델링 결과에 부합되는 데이터를 만들어야 되기 때문에
